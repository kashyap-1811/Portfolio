Featured Project

Artha — Microservices Budget & Expense Tracker

Artha (Sanskrit for wealth/finance) is a full-stack, microservices platform for multi-company budgeting and expense tracking with real-time analytics, alerts, and secure access. Deployed on DigitalOcean behind Nginx/SSL, it combines event-driven services with caching and rate limiting to keep dashboards fast and reliable at scale.

  • Company management with role-based access, personal companies, and fiscal-period budgets with category allocations.
  • Expense approvals, monthly history, and budget progress visuals with thresholds at 80%/100% (SendGrid alerts).
  • Python analytics service (FastAPI) for health scores, category breakdowns, MoM trends, top spenders, and active budget analysis.
  • Event-driven CQRS with Kafka topics for expense, budget, and membership events plus async notification workflows.
  • Redis caching for hot reads, Redis rate limiting at the API gateway, and JWT + Google OAuth authentication.
  • Services routed via Spring Cloud Gateway + Eureka, backed by PostgreSQL, MongoDB Atlas, Docker Compose, and Kafka UI.
React 19 Vite Java 17 Spring Boot Spring Cloud FastAPI Node.js Kafka Redis PostgreSQL MongoDB Atlas SendGrid Docker DigitalOcean Nginx
