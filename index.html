Artha Architecture Map

Artha Architecture

Artha is a high-performance budgeting platform utilizing a microservices architecture of 7 containerized services. It handles routing and authentication via Spring Cloud Gateway with centralized JWT validation, backed by Eureka service discovery, Redis rate limiting, and an event-driven core.

Technical Deep Dive & Specifications

Microservices Architecture

Architected a 7-service containerized system comprising Spring Boot core services, a Node.js notification engine, and a FastAPI analytics service.

  • Routed via Nginx + API Gateway (Spring Cloud).
  • Centralized discovery & dynamic registries with Netflix Eureka.
System Optimization & Streaming

Combined Kafka event-streaming with adaptive Redis rate limiting for high-throughput transactional stability.

  • Transactional Outbox pattern prevents dual-write discrepancies.
  • Reduced service latencies by up to 83% using multi-layer caching.
CI/CD Pipeline

Configured automated GitHub Actions workflows with path-based filtering to push changed services to GHCR.

  • Path-based filtering triggers builds only for modified modules.
  • Zero-downtime rolling updates via health-checked shadow containers.
Infrastructure & Memory

Deployed on DigitalOcean (2CPU, 4GB RAM) using Docker Compose, orchestrating databases and message brokers.

  • Enforced strict 350MB per-service memory allocation limits.
  • Integrates external cloud PostgreSQL (Neon) and MongoDB (Atlas) links.
Frontend & UI

Developed a responsive React dashboard featuring interactive charts (MoM trends, categories) to visualize financial diagnostics.

  • High-fidelity data visualizations rendering real-time budget balances.
  • Optimized asset delivery and gzip compressions through custom Nginx hosting configurations.
